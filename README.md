# 🎓 수시레시피 디자인 시스템 문서

저희 수시레시피는 디자이너와 개발자간의 효율적인 협업을 위해 **atomic design**을 적용하여 디자인 시스템을 구축하였습니다.

> 🏠 [홈페이지](https://susirecipe.github.io/susirecipe-design-system/)를 통해서 디자인 시스템 문서를 확인해보세요!
